Instructions to run the file

Use MySQL workbench, create a new session, and paste the SQL code of the back end given in the textfile name backend.txt

Press the lightning symbol on the workbench

Open terminal. Make sure node.js is installed on the PC

Using the terminal navigate to the folder Raaga containing all the code

Use the command - node index.js 

A server will be created running on port 3000 

Now use any browser. 

Type the following: http://localhost:3000/

Ensure that the port number matches the port on which the server is running. In our case, it is port 3000.

You’ll be directed to the site
